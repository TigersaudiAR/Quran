# تعليمات نشر تطبيق القرآن الكريم التعليمي على Netlify

## الخطوة 1: إعداد المشروع

1. تأكد أن لديك Node.js وnpm مثبتين على جهازك
2. قم بفك ضغط ملف `quran-educational-app.zip` الذي قمت بتنزيله
3. افتح موجه الأوامر (Terminal) وانتقل إلى مجلد المشروع:
   ```
   cd path/to/quran-educational-app
   ```
4. قم بتثبيت اعتمادات المشروع:
   ```
   npm install
   ```

## الخطوة 2: اختبار التطبيق محلياً

1. قم بتشغيل التطبيق على خادم التطوير المحلي:
   ```
   npm start
   ```
2. سيتم فتح التطبيق تلقائياً في المتصفح على العنوان `http://localhost:3000`
3. تأكد من أن التطبيق يعمل بشكل صحيح ويتصل بـ API الخاص بـ alquran.cloud

## الخطوة 3: بناء التطبيق للإنتاج

1. قم ببناء نسخة الإنتاج من التطبيق:
   ```
   npm run build
   ```
2. سيتم إنشاء مجلد `build` يحتوي على ملفات التطبيق الجاهزة للنشر

## الخطوة 4: النشر على Netlify

### الطريقة الأولى: النشر عبر واجهة Netlify الرسومية (الأسهل)

1. انتقل إلى [موقع Netlify](https://app.netlify.com/) وقم بتسجيل الدخول (أو إنشاء حساب جديد)
2. اسحب مجلد `build` الذي تم إنشاؤه في الخطوة السابقة وأفلته في المنطقة المخصصة للسحب والإفلات في واجهة Netlify
3. انتظر حتى يتم رفع الملفات ونشر الموقع (عادة ما يستغرق بضع دقائق)
4. ستحصل على عنوان URL للموقع المنشور (مثال: `https://your-app-name.netlify.app`)

### الطريقة الثانية: النشر باستخدام Netlify CLI

1. قم بتثبيت أداة Netlify CLI:
   ```
   npm install -g netlify-cli
   ```
2. سجل الدخول إلى حسابك في Netlify:
   ```
   netlify login
   ```
3. قم بإعداد مشروع جديد:
   ```
   netlify init
   ```
   - اختر "Create & configure a new site"
   - اختر الفريق (Team)
   - أدخل اسماً للموقع (اختياري)
4. قم بنشر التطبيق:
   ```
   netlify deploy --prod
   ```
   - عندما يُطلب منك تحديد مجلد النشر، أدخل `build`

## الخطوة 5: تكوين إعدادات النشر (إذا لزم الأمر)

1. انتقل إلى لوحة تحكم Netlify الخاصة بموقعك
2. في قسم "Site settings" > "Build & deploy" > "Continuous Deployment":
   - تأكد من وجود ملف `_redirects` في مجلد `public` للتعامل مع توجيهات React Router
3. تحت "Domain management"، يمكنك:
   - تخصيص اسم النطاق الفرعي لـ Netlify
   - إضافة نطاق مخصص إذا كان لديك

## ملاحظات هامة

1. يستخدم التطبيق واجهة برمجة التطبيقات (API) من alquran.cloud، وهي خدمة مجانية قد تكون لها حدود للاستخدام
2. تأكد من أن CORS معد بشكل صحيح للسماح بالاتصال من النطاق المستضاف
3. الملفات الصوتية تأتي من CDN خاص بـ islamic.network
4. التطبيق يدعم وضع عدم الاتصال جزئياً للآيات والترجمات المحفوظة، ولكن تشغيل الصوت يتطلب اتصالاً بالإنترنت

## المحتويات الرئيسية للتطبيق

- قراءة القرآن الكريم بتنسيق الرسم العثماني
- واجهة لعرض السور أو الصفحات
- الاستماع إلى الآيات بأصوات مختلف القراء
- ترجمات متعددة بلغات مختلفة
- البحث في نص القرآن الكريم
- إضافة إشارات مرجعية وحفظ آخر قراءة
- تعلم التجويد والدروس التفاعلية
- حلقات التحفيظ والفصول الافتراضية

## رابط التطبيق بعد النشر

بعد اكتمال عملية النشر، يمكنك الوصول إلى التطبيق عبر الرابط الذي يوفره Netlify:
`https://[your-site-name].netlify.app`

يمكنك مشاركة هذا الرابط مع الآخرين للوصول إلى التطبيق.
